# Aetheris Social Poster

AI-powered social media automation tool that posts humanized content to Reddit, X (Twitter), LinkedIn, and Facebook twice daily.

## Features

- **AI Content Generation**: Uses Claude API to create unique, humanized content for each platform
- **AI Image Generation**: Automatic image creation using Pollinations.ai (free)
- **Browser Automation**: Puppeteer-based posting that works with your logged-in sessions
- **Multi-Platform Support**: Reddit, X (Twitter), LinkedIn, and Facebook
- **Scheduled Posting**: Automatic twice-daily posting at configurable times
- **Analytics Dashboard**: Track posting history, success rates, and platform performance
- **Web Dashboard**: Modern React UI for content management and monitoring

## Tech Stack

- **Backend**: Node.js, Express, Puppeteer
- **Frontend**: React, CSS
- **AI**: Claude API (Anthropic), Pollinations.ai (images)
- **Scheduling**: node-cron
